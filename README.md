## Setup

This project is built using [Nest](https://github.com/nestjs/nest) framework.

Watch: https://youtu.be/0M8AYU_hPas

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

To access the app start [localhost:3000](http://localhost:3000)

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
